## в—Ҳ Overview

A fully local **Retrieval-Augmented Generation (RAG)** chatbot built as part of an **AI Advanced Skill Development Lab**. It runs entirely on your machine вҖ” no OpenAI, no paid APIs, no internet required after setup.

Upload **PDF** or **TXT** documents and ask questions about them. The chatbot splits your documents into chunks, stores them as vector embeddings in **ChromaDB**, and retrieves the most relevant context before passing it to **Gemma** (via Ollama) to generate accurate, grounded answers.

> Built with FastAPI В· LangChain В· ChromaDB В· and a custom **purple hacker-terminal UI**. Dockerized for easy deployment.

## в—Ҳ Getting Started

### Prerequisites
- [Ollama](https://ollama.com) installed and running
- [Docker](https://docker.com) installed (optional)
- Python 3.11+

### 1 В· Pull the model

```bash
ollama pull gemma:2b
```

### 2 В· Install dependencies

```bash
cd backend
python -m venv venv

# Windows
.\venv\Scripts\Activate.ps1

# Mac / Linux
source venv/bin/activate

pip install -r requirements.txt
```

### 3 В· Run the server

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 4 В· Open the chatbot

Navigate to **http://localhost:8000** in your browser.

---

## в—Ҳ Docker Setup

```bash
# Build and start everything
docker-compose up --build

# Stop
docker-compose down
```

> **Linux users:** If `host.docker.internal` fails, set `network_mode: host` in `docker-compose.yml` and use `OLLAMA_BASE_URL=http://localhost:11434`

## в—Ҳ Usage

1. Start Ollama вҖ” `ollama serve`
2. Start backend вҖ” `uvicorn main:app --port 8000 --reload`
3. Open `http://localhost:8000`
4. **Drag and drop** a PDF or TXT file into the sidebar
5. Wait for the `MODE: RAG` indicator to activate
6. Ask questions about your document
